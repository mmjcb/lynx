# Lynx - All Your Links in One Place

Lynx is a web application that allows users to organize, manage, and share their links in a secure and easy-to-use platform. Create rooms, add links, and collaborate with others seamlessly.

---

## Features

- **User Authentication**  
  Sign up, sign in, and sign out securely using Firebase Authentication.

- **Room Management**  
  - Create, edit, and delete rooms.  
  - Join rooms using a room code.

- **Link Management**  
  - Add, edit, and delete links inside each room.  
  - View a list of links with quick access to open them.

- **Responsive UI**  
  Works on both desktop and mobile browsers.

---

## Technology Stack

- **Frontend**: HTML, CSS, JavaScript  
- **Backend / Database**: Firebase Firestore  
- **Authentication**: Firebase Auth  
- **Hosting**: Vercel
